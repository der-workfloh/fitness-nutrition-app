wsCatalyst

Allgemeines

  • Applikations-Entwicklungs-System
  • Frameworkkernel bildet das APF
  • Compilation zur Codeoptimierung und Datenreduzierung

Features

Erweitert das Kernel-Framework um viele nützliche Funktionen

  • xMail - Umfangreiche, konfigurierbare Mailing-Funktion
  • extIterator - Erweiterte Iteratoren-Funktionalitäten
  • Integriert Benutzerverwaltung in das Template-System (basierend auf dem Usermanagement-Modul)
  • Erweiterte Datenbank-Klasse für PDO und MongoDB
  • Erweiterte Logging-Möglichkeiten
  • Klassen zum Handeln von Datum und diversen Datum-Formaten
  • Unterstützt Unittests
  • Integrierte System-Message-Funktionalität
  • RouterHandler vereinfacht das Arbeiten mit der URL
  • Ein umfangreiches Template-System ermöglicht diverse Ansprecharten (Normal, Ajax, Cronjob, ... )
  • Integrierte Http-Komprimierung
  • Gettext-Unterstützung für flexible multilinguale Seiten
  • ...
